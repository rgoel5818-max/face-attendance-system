# Face Recognition Attendance System

Python + OpenCV based attendance system using face recognition.

## Features
- Real-time face detection
- Automatic attendance marking
- CSV export with date/time
- Multiple student support

## Tech Stack
- Python
- OpenCV
- face_recognition
- NumPy
- Pandas
